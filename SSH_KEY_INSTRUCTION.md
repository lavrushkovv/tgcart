# Инструкция по добавлению SSH-ключа в GitHub

## Шаг 1: Скопируйте публичный ключ

В терминале выполните:
```bash
cat ~/.ssh/id_rsa_new.pub
```

Скопируйте всю строку, начиная с `ssh-rsa` и заканчивая вашим email.

## Шаг 2: Добавьте ключ в GitHub

1. Перейдите на [GitHub](https://github.com)
2. Нажмите на ваш аватар в правом верхнем углу → **Settings**
3. В левом меню прокрутите вниз до **SSH and GPG keys**
4. Нажмите **New SSH key**
5. В поле **Title** введите, например: `MacBook-Viktor`
6. В поле **Key** вставьте скопированный ключ
7. Нажмите **Add SSH key**
