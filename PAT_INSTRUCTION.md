# Инструкция по созданию Personal Access Token (PAT) для GitHub

## Шаг 1: Перейдите на страницу генерации токена

Откройте [https://github.com/settings/tokens](https://github.com/settings/tokens)

## Шаг 2: Создайте новый токен

1. Нажмите **Generate new token** → **Generate new token (classic)**
2. В поле **Note** введите, например: `tgcart-push`
3. Установите срок действия (например, 30 дней или 90 дней)
4. Выберите права (scopes):
   - ✅ `repo` — полный доступ к репозиториям (включает push)
5. Нажмите **Generate token**

## Шаг 3: Скопируйте токен

⚠️ **Важно**: Скопируйте токен сразу, после закрытия страницы он будет утерян!

## Шаг 4: Используйте токен при push

После копирования токена, выполните в терминале:

```bash
git push -u origin main
```

При запросе имени пользователя введите: `lavrushkovv` (или ваш GitHub username)
При запросе пароля вставьте скопированный токен.

---

**Альтернативно (рекомендуется)**: Вы можете сохранить токен в системе менеджера паролей macOS, чтобы не вводить его каждый раз.
