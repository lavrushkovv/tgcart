# 🛠️ Инструкция по запуску Admin Panel

## 📋 Предварительные требования

- Node.js 16+ и npm
- Telegram бот с токеном
- ADMIN_CHAT_ID настроен в `.env` файле server

## 🚀 Быстрый старт

### 1. Сервер

```bash
cd server
npm install
# Создайте .env файл (см. ниже)
npm run start
```

**server/.env файл:**
```
BOT_TOKEN=ваш_токен_бота
CLIENT_APP_URL=https://tgcart.vercel.app
ADMIN_CHAT_ID=123456789  # ваш Telegram ID
ADMIN_PANEL_URL=https://your-admin-panel-url.com  # URL вашей админ-панели
PORT=5000  # необязательно, по умолчанию 5000
```

### 2. Admin Panel

```bash
cd admin-panel
npm install
npm run dev
```

Панель будет доступна на http://localhost:5173

### 3. Деплой на Vercel/Netlify

```bash
cd admin-panel
npm run build
```

Загрузите папку `admin-panel/dist` на ваш хостинг.

## 🔐 Как использовать

1. Откройте Telegram и найдите вашего бота
2. Нажмите на кнопку меню (или отправьте любое сообщение)
3. Нажмите кнопку **"Admin Panel"**
4. Выполните вход через Telegram (автоматически проверяются права администратора)
5. Готово! Теперь вы можете:
   - 📦 Просматривать все заказы и обновлять их статус
   - 🛍️ Управлять продуктами (добавлять, редактировать, удалять)

## 📊 Меню статусов заказов

- **new** - Новый заказ (нужно связаться с покупателем)
- **confirmed** - Подтвержден
- **shipped** - Отправлен
- **completed** - Завершен
- **cancelled** - Отменен

## 🔌 API эндпоинты

Все эндпоинты админ-панели требуют заголовок `X-Telegram-Auth` с данными Telegram Mini App.

| Метод | Путь | Описание |
|-------|------|----------|
| POST | /admin/login | Аутентификация администратора |
| GET | /api/orders | Получить все заказы |
| PUT | /api/orders/:id/status | Обновить статус заказа |
| GET | /api/products | Получить все продукты |
| POST | /api/products | Добавить новый продукт |
| PUT | /api/products/:id | Обновить продукт |
| DELETE | /api/products/:id | Удалить продукт |

## 🐛 Отладка

### Если админ-панель не открывается:

1. Проверьте ADMIN_CHAT_ID в server/.env (должен совпадать с вашим Telegram ID)
2. Убедитесь, что вы начали диалог с ботом
3. Проверьте консоль браузера на ошибки
4. Проверьте логи сервера

### Если не удается добавить продукт:

1. Убедитесь, что все обязательные поля заполнены
2. Проверьте формат цены (число с двумя знаками после точки)
3. Убедитесь, что URL картинки валидный

## 📝 Примечания

- Админ-панель автоматически разворачивается на весь экран при запуске
- Все изменения применяются мгновенно
- Приложение использует нативную тему Telegram
- Поддерживается Haptic feedback (вибрация)
