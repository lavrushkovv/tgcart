# План настройки клиентской части (Frontend) - STREET APPLE

## Обзор
Настройка React-приложения для Telegram Mini App магазина STREET APPLE

## Шаги выполнения

### 1. Установка зависимостей client
Перейти в папку client и установить все необходимые пакеты:
```bash
cd client
npm install
```

### 2. Обновление конфигурации Vite
Обновить файл `client/vite.config.js` для добавления переменной окружения VITE_APP_NAME с названием магазина.

### 3. Создание .env файла
Создать файл `client/.env` с необходимыми переменными окружения:
```
VITE_APP_BACKEND_URL=https://your-backend-url.com
VITE_APP_NAME=STREET APPLE
```

### 4. Обновление index.html
Обновить заголовок в `client/index.html` на "STREET APPLE - E-commerce" и добавить описание магазина.

### 5. Проверка работоспособности
Запустить dev-сервер для проверки:
```bash
npm run dev
```

## Ожидаемые результаты
- Все зависимости установлены
- Приложение настроено с названием STREET APPLE
- Dev-сервер запускается без ошибок
- Приложение готово к интеграции с backend сервером
